## 9. LINKEDIN POST — Settimana 3: Tauri 2.0

```
Why I chose Tauri 2.0 over Electron — and why it matters for AI desktop apps 🖥️

VIO 83 AI Orchestra is a desktop app. Not a web app. Not a PWA. A real, native desktop application.

But I didn't use Electron. Here's why:

Electron app size: 100-200 MB
Tauri app size: 2 MB

Electron RAM usage: 300-500 MB
Tauri RAM usage: 30-50 MB

Electron = ships entire Chromium browser
Tauri = uses native WebView (already on your Mac)

For an AI app that needs to run Ollama models locally (which already consume 2-5 GB RAM), every MB of RAM matters. Especially on my M1 with 8GB.

With Tauri 2.0:
• Rust backend → memory-safe, blazing fast
• Native WebView → no bloat
• 2MB app → instant download
• macOS Keychain → hardware-encrypted API key storage
• Shell plugin → direct Ollama integration

The result: a desktop AI orchestrator that's lighter than Slack.

Tech decisions matter. I chose the hard path because it's the right one.
```
